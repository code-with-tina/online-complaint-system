## 📌 Project Description

The **Online Complaint Management System** is a web-based application developed using **PHP and MySQL** that allows users to register complaints online and track their status efficiently.

It provides an **admin dashboard** to manage, review, and update complaint statuses, improving transparency and reducing manual workload in complaint handling systems.

This system is designed to digitize complaint registration and resolution workflow in organizations, making it faster, organized, and user-friendly.

---

## ⚙️ Features

- 👤 User Registration & Login
- 📝 Submit Complaints Online
- 📊 Admin Dashboard
- 🔄 Complaint Status Tracking (Pending / In Progress / Resolved)
- 🗑️ Admin can Delete / Update complaints
- 🔐 Secure Login System

## 🚀 How to Run the Project

### 1️⃣ Install Requirements
- XAMPP Server
- VS Code (optional)

---

### 2️⃣ Setup Project
- Copy project folder to:
- C:\xampp\htdocs\complaint_system


---

### 3️⃣ Start Server
- Open XAMPP
- Start:
  - Apache
  - MySQL

---

### 4️⃣ Import Database
- Open phpMyAdmin
- Create database (e.g. `complaint_system`)
- Import `.sql` file from project folder

---

### 5️⃣ Run Project
Open browser:


http://localhost/complaint\_system
